# GiáĐất.VN - Hệ thống tra cứu giá đất

Dự án cung cấp hệ thống tra cứu giá đất tại TP.HCM với giao diện bản đồ tương tác, giúp người dùng dễ dàng tra cứu giá đất theo vị trí, tên đường hoặc tọa độ.

## Tính năng chính

- Bản đồ tương tác hiển thị giá đất
- Tìm kiếm theo tên đường và quận
- Tìm kiếm theo tọa độ
- Hiển thị thông tin chi tiết về giá đất
- Hệ thống tài khoản người dùng

## Công nghệ sử dụng

- HTML/CSS/JavaScript
- Bootstrap 5
- Leaflet JS (bản đồ)
- Flask (backend - đang phát triển)

## Hướng dẫn sử dụng

1. Mở trang web
2. Tìm kiếm thông tin qua thanh công cụ bên trái
3. Hoặc click trực tiếp lên bản đồ để xem thông tin tại vị trí đó

## Cách cài đặt và chạy dự án

1. Clone repository về máy
   ```
   git clone https://github.com/MinTieeeen/flask-giadat.git
   ```

2. Mở thư mục dự án
   ```
   cd flask-giadat
   ```

3. Mở file index.html bằng trình duyệt hoặc tạo máy chủ web local
